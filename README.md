# Gabble
---
<br/>
Fullstack Twitter-like page (Mongoose, Node.js, Express, Mustache)
<br/>
To Run: <br/>
  git clone <URL> <br/>
  run npm install <br/>
  (in Command line) type: node app.js <br/>
  go to "http://localhost:5000/"
